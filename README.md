# 📝 PyQt5 Note App

A simple desktop note-taking application built with **Python** and **PyQt5**.  
This app allows users to create, update, delete, and manage notes through a clean graphical interface.

## 🚀 Features

- Add new notes with title and description
- Update existing notes
- Delete notes
- Select notes from a list to edit
- Clear selection and reset the form
- Simple GUI built with PyQt5
- Unique note IDs using UUID

## 🧠 How It Works

The project is divided into two main parts:

1. **Logic Layer (NoteManager)**
   - Handles note storage and management
   - Supports add, update, delete, and retrieve operations

2. **UI Layer (NoteApp)**
   - Built with PyQt5
   - Displays notes in a list
   - Provides a form to add or edit notes
   - Connects UI actions to the logic layer

## 🛠 Technologies Used

- Python 3
- PyQt5
- UUID (for generating unique note IDs)
